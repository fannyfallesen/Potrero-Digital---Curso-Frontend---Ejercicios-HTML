Ejercicio 7 - Probando Etiquetas varias

En este Ejercicio voy a probar las etiquetas de cambio de formato de texto, las cuales son: Negrita, Cursiva y Subrayado.
Tambien voy a probar otras etiquetas como Salto de linea (< br> o < br/>), agregar imagenes (<img src=""/>), Formulario(<form></form>), junto con label(<label></label>), input (<input></input>), textarea que es para el ingreso de multilineas, por ejemplo un comentario (<textarea></textarea>) y button (<button></button>).
